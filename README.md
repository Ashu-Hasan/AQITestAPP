# 🌿 AQI Monitoring Android App

An Android application that displays real-time Air Quality Index (AQI) data with visually appealing custom graphs and smooth user experience.

---

## 🚀 Features

* 📡 **Live AQI Data Integration**
  Fetches real-time AQI data from API and updates dynamically.

* 📊 **Custom Graphs & Charts**
  Built custom visual graphs to represent AQI trends in an intuitive way.

* 🎨 **Modern UI (Material Design)**
  Clean and responsive UI with Material Design components.

* 🌙 **Dark Mode Support**
  Separate night theme support for better user experience.

* 📍 **Location-Based AQI**
  Displays AQI based on selected or current location.

* ⚡ **Efficient Data Handling**
  Optimized API calls and UI updates for smooth performance.

---

## 🛠️ Tech Stack

* **Language:** Java
* **Networking:** Retrofit
* **UI:** XML, Material Components
* **Charts:** Custom graph implementation
